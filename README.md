# Immich TikTok Remover

Home Assistant add-on port of [immich-tiktok-remover](https://github.com/mxc2/immich-tiktok-remover) (stable-lite: filename/date detection only, no ML).

## Installation

1. Add this repository in the Supervisor add-on store:

   ```text
   https://github.com/mxc2/immich-tiktok-remover-HAOS
   ```

2. Install **Immich TikTok Remover**, configure your Immich URL/API key under **Scan jobs**, then start the add-on.

See [DOCS.md](DOCS.md) for full configuration details.
